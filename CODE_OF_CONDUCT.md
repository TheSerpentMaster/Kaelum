# Code of Conduct

## Our Standard

This project expects respectful, constructive collaboration.

Examples of behavior that contribute to a healthy environment:

- Using welcoming and professional language
- Accepting feedback in good faith
- Focusing discussion on the code, behavior, and design rather than the person
- Being clear about tradeoffs, risks, and assumptions
- Respecting different levels of experience

Examples of unacceptable behavior:

- Harassment, threats, or personal attacks
- Discriminatory or degrading language
- Deliberate hostility, trolling, or baiting
- Publishing private information without permission
- Disruptive conduct that makes collaboration harder

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, pull requests, commits, or other contributions that violate this Code of Conduct.

Repeated or severe violations may result in temporary or permanent exclusion from the project.

## Reporting

If you experience or witness unacceptable behavior, contact the project maintainers through the repository hosting platform and include relevant context.

## Project Focus

This repository is a technical collaboration space. Keep discussion relevant to the project and aim for direct, respectful problem-solving.
