# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.1.2   | :white_check_mark: |
| < 1.1   | :x:                |

## Reporting a Vulnerability

When you find a vulnerability, we highly recommend you post it under the repo, in the comments section. Our developers will be investigating the issue shortly after you post it.
